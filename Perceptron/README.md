# Perceptron simples 

Perceptron de camada única, ele lê um valor com 7 nós de entrada e retorna 1 caso esse valor seja maior que o dado de controle (RU), caso contrário, retorna -1.

O treinamento define os pesos para cada nó, a saída é:

```
Pesos originais:  [1. 1. 1. 1. 1. 1. 1.]
Novos pesos: [ 1.08  0.02  0.1   0.02 -0.14 -0.08 -0.02]
========================================================
Erro amostral total: 19
```
